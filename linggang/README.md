# 人员全生命周期一体化管控平台 - 产品原型

## 项目概述
基于「人员全生命周期一体化管控平台」完整需求规格说明书设计并实现的全模块产品原型。

## 技术栈
- **前端框架**: Vue 3 + Element Plus 3.x (CDN)
- **UI 风格**: 严格对齐开源项目 pig-ui（基于 Element-UI 3.x）
- **图表/流程图**: CSS + JavaScript 原生实现 BPMN 模拟
- **兼容性**: Chrome / Firefox / Edge（PC端 1920×1080+）

## 快速启动
```bash
# 方式1：直接打开
open index.html

# 方式2：使用本地服务器（推荐）
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## 核心模块对应关系

### 1. 工作流模块（核心重点）
| 二级菜单 | 三级菜单 | 菜单标识 |
|---------|---------|---------|
| 准入办证服务（A模块） | 已发/待办/已办任务 | workflow-[issued/pending/done]-a |
| 访客管理服务（B模块） | 已发/待办/已办任务 | workflow-[issued/pending/done]-b |
| 离场准出服务（C模块） | 已发/待办/已办任务 | workflow-[issued/pending/done]-c |
| 安全考评服务（D模块） | 已发/待办/已办任务 | workflow-[issued/pending/done]-d |

**工作流任务编辑器页面包含**：
- **表单 Tab**: 各业务模块对应的申请表单（字段对齐需求文档表结构）
- **流程图 Tab**: BPMN 2.0 流程图可视化（CSS模拟），高亮当前/已完成节点
- **流转记录 Tab**: 时间轴展示全量流转记录

### 2. 组织权限服务
- 组织管理（树形结构）
- 角色管理（权限配置）
- 用户管理（含脱敏处理）
- 菜单权限（按钮级权限配置）

### 3. 人员主数据服务
- 一人一档（个人信息卡片 + 列表）
- 资质管理（资质有效期预警）
- 培训记录
- 安全记录（事故/违章）

### 4. 在岗现场管控服务
- 违规事件列表（统计卡片 + 表格）
- 工单闭环管理
- 电子围栏告警（地图占位 + 围栏列表）
- 定位轨迹（地图占位）

### 5. 安全考评积分服务
- 积分规则配置（卡片式规则列表）
- 积分计算（Q1/Q2周期计算）
- 积分管理（积分进度条 + 联动冻结）

### 6. 公共能力服务
- 文件上传/下载管理
- 消息推送（系统/流程/预警/告警通知）
- 日志审计（操作日志记录）

## 交互特性
- **脱敏处理**: 手机号、身份证默认脱敏展示，点击需确认（操作记录至审计日志）
- **联动场景**: 模拟「资质过期冻结权限」「积分低分冻结证件」场景
- **响应速度**: 提交操作模拟 ≤1s 响应，导出模拟加载状态
- **流程操作**: 下一步/抄送/驳回/终止/保存草稿（按钮权限按流程节点适配）
- **流程图交互**: 支持缩放，节点悬停显示处理人/时间/意见

## 代码规范
- HTML: 语义化标签，Element Plus 组件风格
- CSS: BEM 命名，CSS 变量复用 pig-ui 配色体系
- JS: 模块化拆分，关键逻辑中文注释
- 无控制台报错
